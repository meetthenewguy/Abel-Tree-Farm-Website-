# Phase 2 SEO and Availability Plan

## Availability portal

Current live implementation:
- Header and inventory area link to `availability.html`.
- The page is intentionally honest: it does not claim a live inventory file exists yet.
- Buyers are directed to call, text, or email with plant names, sizes, quantities, timing, and project location.

Recommended next implementation:
1. Create a Google Sheet or PDF named `Abel Tree Farm Weekly Availability`.
2. Keep columns simple: plant, botanical name, size, quantity, notes, last updated.
3. Publish or share the file with a stable public link.
4. Replace the temporary availability-page language with a direct `Download Current Availability` button.
5. Update the file weekly and include a visible `Last updated` date.

## Domain email

Do not replace the visible AOL email on the public website until a domain address is created and tested.

Recommended address options:
- `info@abeltreefarm.com`
- `sales@abeltreefarm.com`

Required before site update:
1. Create domain mailbox or forwarder.
2. Forward to `abeltreefarm@aol.com`.
3. Send inbound and outbound tests.
4. Confirm the farm receives messages reliably.
5. Replace all visible AOL references and structured data email values.

## SEO landing pages

Recommended first plant pages:
- `/inventory/green-buttonwood.html`
- `/inventory/areca-palms.html`
- `/inventory/podocarpus.html`
- `/inventory/seagrape.html`
- `/inventory/green-island-ficus.html`

Each page should include:
- Plant-specific H1.
- Local South Florida use cases.
- Current availability CTA.
- Call/text buttons.
- Honest availability caveat.
- Internal links back to full inventory.

Do not create live plant pages until the farm confirms that each plant is a priority product and wants that page indexed.
